1.什么是JavaScript？和Java有关系吗

java和javascript没有任何关系。java运行在JVM当中，而javascript运行在浏览器当中 javascript以前叫作livescript，是网景公司“布兰登艾奇”开发的，在历史的某个阶段， SUN公司和网景公司有过合作，SUN公司将livescript改名为javascript，蹭java的热度，搭上java的顺风车 经过多年的发展，也应证了当年的做法是正确的

2.JavaScript的语法和Java有些类似

3.JavaScript是一门脚本语言

所有的脚本语言都有这样一个特点：脚本语言最终运行的那个程序是以普通文本的形式保存的。 java就不是脚本语言，java语言需要先编写xxx.java源文件，在源文件中编写java源代码， 然后java源代码还需要进行一个编译，编译之后会生成xxx.class文件，而最终运行的是class文件 class文件不是普通文本，用记事本打不开，所有java不是脚本语言。javascript是脚本语言

4.JS和JSP有什么关系

JS和JSP也是没有任何关系的。 JS是：Javascript的缩写 JSP是：Java Server Pages的缩写。（JSP实际上还是java程序）

5.JS是干嘛的

HTML CSS JS它们的关系？ HTML是主体。 CSS是用来修饰HTML的。 JS是用来操作HTML和CSS的，修改HTML中的元素，修改CSS的样式，达到网页的动态效果
